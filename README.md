# decode 104 telegram
Script to decode IEC 60870-5-104 telegrams


run main.py and past telegrams  
```python main.py```  

If you need show less information  
```python main.py --less```

If you need print only APCI run  
```python main.py --head```  
If you need print only ASDU run  
```python main.py --body```
